# Goalsetter MERN app

This is the goalsetter app from the [Learn the MERN Stack](https://www.youtube.com/watch?v=-0exw-9YJBo) series on YouTube.

## Usage

Rename the .envexample to .env and add your MONGO_URI

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```

## Libs

### FrontEnd

1. React.js
2. Redux Toolkit

### BackEnd

1. Express
2. Mongoose
3. JWT
